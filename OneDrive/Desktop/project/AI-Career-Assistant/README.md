# 🤖 AI Career Assistant

An AI-powered web application that helps job seekers analyze resumes, identify skill gaps, generate professional resumes, and prepare for interviews.

Built using **Python, Streamlit, and Google's Gemini API**, this tool provides intelligent career insights based on resume content.

---

## 🚀 Features

### 📄 Resume Analyzer

Upload a resume (PDF) and get AI-powered feedback:

* Resume score
* Skills detected
* Missing skills
* Suggestions for improvement
* Recommended career roles

### 📊 Skill Gap Analysis

Displays a visual chart comparing **existing skills vs missing skills**.

### 📝 AI Resume Builder

Generate a professional resume using:

* Name
* Skills
* Education
* Experience / Projects

### 🎤 Interview Preparation

Generate interview questions including:

* Technical questions
* Behavioral questions
* HR questions

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* pdfplumber
* pandas

---

## ▶️ Run the Project

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python -m streamlit run app.py
```

Open in browser:

```
http://localhost:8501
```
